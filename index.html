Projects

Here you will find some of the personal and clients projects that I created with each project containing its own case study

NestApp

I recreated the frontend of a modern ECommerce web app inspired by top platforms. The clean product grid, smooth transitions, and responsive design really caught my attention, so I took the challenge to rebuild the UI from scratch using React.js and firebase . This project helped me explore dynamic filtering, shopping cart interactions, and mobile-first layouts — all while maintaining a seamless and polished shopping experience. It was a great hands-on opportunity to understand how real-world ECommerce platforms structure their frontend for usability and conversion.

LinkedIn's CloneApp

I built a fully responsive LinkedIn UI clone to sharpen my frontend and state management skills. I was particularly drawn to LinkedIn's clean feed layout, profile structure, and the intuitive flow between pages. Using React, Firebase Auth, and Redux Toolkit, I recreated key features like login, post feeds, and profile sections. This project taught me how social platforms handle user sessions, conditional rendering, and component-based architecture. Rebuilding LinkedIn’s design from the ground up helped me get comfortable with scalable UI logic and user-focused design.

Holiday Homes

I recreated the frontend of a modern Holiday Homes booking platform, inspired by top travel and accommodation websites. The clean property listing grid, smooth page transitions, and fully responsive design stood out to me, so I challenged myself to rebuild the UI from scratch using React.js and Firebase. This project allowed me to dive deep into dynamic filtering.mobile-first layouts It was a valuable hands-on experience that helped me understand how leading booking platforms structure their frontends for performance, usability, and user conversion.
